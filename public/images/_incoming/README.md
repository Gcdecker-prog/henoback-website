# Image drop folder

Place new downloads here with any filename. Then ask the agent to run the sort step, or copy manually into:

- `brand/` — logo
- `hero/` — homepage hero
- `services/{slug}.png` — slug matches `lib/content/services.ts`
- `industries/{slug}.png` — slug matches `lib/content/industries.ts`
- `team/` — headshots (`jim-french.png`, etc.)
- `case-studies/` — case study heroes
- `partners/` — partner logos

All paths are registered in `lib/content/media.ts`.
